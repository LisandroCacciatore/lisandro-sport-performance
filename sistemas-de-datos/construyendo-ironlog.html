IronLog: sistema de datos para entrenadores de fuerza con múltiples atletas

El mismo problema que existe en clubes de fútbol — datos capturados sin proceso — aparece en entrenamiento de fuerza cuando un entrenador coordina 20 o más atletas. IronLog resuelve ese problema específico: estructura el registro, calcula e1RM actualizado y alerta cuando el progreso de un atleta se detiene.

Lisandro Cacciatore · Acceso actual por lista de espera

El problema es el mismo, el contexto es diferente

Trabajando con entrenadores de powerlifting y halterofilia que gestionan entre 20 y 50 atletas, encontré un patrón idéntico al que veo en los clubes de fútbol: todos tienen datos, nadie tiene proceso.

Los registros de entrenamiento llegan por WhatsApp. Los archivos de Excel se rompen cuando alguien agrega un atleta. Generar un reporte de progreso para la semana lleva horas. Y la decisión de si un atleta necesita ajuste de carga se toma "a ojo", porque organizar la información lleva más tiempo que analizarla.

El problema no es que los entrenadores no quieran trabajar con datos. El problema es que ninguna herramienta fue diseñada específicamente para este flujo de trabajo.

Qué resuelve IronLog

IronLog es una plataforma pensada para entrenadores de fuerza que coordinan múltiples atletas. No es un GPS adaptado. No es una planilla mejorada. Es un sistema construido desde las preguntas que los entrenadores se hacen realmente:

  • ¿Quién está progresando y quién se estancó? El dashboard muestra la trayectoria de cada atleta en los movimientos principales, con alertas cuando el progreso se detiene más de dos semanas.
  • ¿Este atleta completó lo planificado hoy? El logging mobile pre-carga lo planificado. El atleta solo registra si hubo cambios. Treinta segundos por sesión.
  • ¿Cuándo fue la última vez que ajusté la carga de este atleta? El sistema lleva el historial completo y calcula e1RM actualizado con fórmulas validadas.

Estado actual: Piloto Cerrado

Actualmente IronLog se encuentra en fase de piloto cerrado. Estoy trabajando con un grupo reducido de entrenadores para validar el flujo de trabajo real antes de abrir la plataforma al público.

Mientras tanto, liberé una serie de simuladores gratuitos (Wendler, Starting Strength, Cube) para que puedas probar la lógica de cálculo que usamos internamente. Preguntas frecuentes

¿Para qué tipo de entrenador es IronLog? +

Para entrenadores de fuerza que coordinan entre 20 y 50 atletas simultáneamente y necesitan estructura para seguir el progreso individual sin que la administración consuma el tiempo de análisis. No es para atletas individuales ni para quienes ya tienen un sistema consolidado.

¿Qué hace IronLog que no puede hacer Excel? +

Calcula e1RM actualizado con fórmulas validadas sin intervención manual, muestra alertas de estancamiento cuando el progreso se detiene más de dos semanas, y permite registro mobile en 30 segundos. El equivalente en Excel requiere horas semanales de administración.

¿En qué estado de desarrollo está IronLog? +

Piloto cerrado. Un grupo reducido de entrenadores está validando el flujo de trabajo real antes de la apertura pública. Los simuladores gratuitos de Wendler, Starting Strength y Cube están disponibles para probar la lógica de cálculo. La lista de espera para acceso prioritario está abierta.
